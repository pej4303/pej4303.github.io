
## 1. 사이드바 배경 이미지 변경
+ `commons.scss` 파일 수정
```
파일 위치 : _sass/addon/commons.scss
```
+ `#sidebar` 검색 해서 아래의 3줄 추가
```
background: url('/assets/img/background.jpg');
background-size: 100% 100%;
background-position: center;
```

## 2. 사이드바 메인 이미지 변경
+ `_config.yml` 파일 수정
+ `avatar` 부분 수정
    ```
    avatar: "/commons/avatar.jpg"
    ```
    + 외부 이미지 링크를 넣어도됨
